When would you want to use a remote repository rather than keeping all your work
local?

    When I work with others on the same project, or when I want share my project with others to use.

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    Because a pull might cause a conflict on the local repository, so it's better to be done manually to be ready to fix the conflict in case it happened.
